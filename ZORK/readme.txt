ZORK I adventure game (see the manual)

Must be played on a RC2014 provided with a 512 MB RAM + 512 MB RAM board and a SIO/2 board (e.g. SC110)
If a Digital I/O board is present, the current (executing) overlay index will be displayed on the leds

ROM512.hex - store-it (burn-it) to the 512 module's EPROM
zork.hex - the ZORK.COM file
zorkcomm.hex - store-it as a file on the same disk as ZORK.COM
